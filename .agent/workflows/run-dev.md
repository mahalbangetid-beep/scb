# Running DICREWA Development Server

## Prerequisites
- Node.js 20+ installed
- npm installed

## Steps

1. Navigate to project directory
```bash
cd c:\Users\chand\Music\v
```

// turbo
2. Install dependencies (if not already installed)
```bash
npm install
```

// turbo
3. Start development server
```bash
npm run dev
```

4. Open browser at http://localhost:5173

## Notes
- The dev server supports hot reload
- All changes to React components will auto-refresh
- CSS changes also hot reload
- Check terminal for any compilation errors
